# FlashTrendy Admin Guide

This guide is for store administrators to manage products, orders, and view store performance.

## Dashboard Overview

The Admin Dashboard provides a real-time snapshot of your store:
- **Total Revenue**: Cumulative earnings from all orders.
- **Total Orders**: Count of orders placed.
- **Total Customers**: Number of registered users.
- **Sales Analytics**: Visual charts showing revenue trends and product performance.

## Product Management

- **Listing**: View all products, search, and filter by status or category.
- **Adding Products**: Use the "New Product" button. You can upload multiple images, set prices, SKU, and manage stock levels.
- **Editing**: Click on any product to update its details or images.
- **Inventory**: Monitor "Low Stock Alerts" to ensure popular items are restocked.

## Order Management

- **Order List**: View all system orders and their current status.
- **Processing**: Click on an order to view customer details and shipping info.
- **Status Updates**: Manually update order status (Pending -> Processing -> Shipped -> Delivered).
- **Cancellations/Refunds**: Manage order cancellations and initiate refunds through the order detail page.

## User Management

- **User Table**: View all registered users.
- **Roles**: Upgrade/Downgrade user roles (e.g., promote a user to ADMIN).
- **Account Actions**: Block/Unblock users or delete accounts if necessary.

## Reporting

- **Export**: Export product, order, or customer data to CSV for external analysis.
- **Analytics**: Deep dive into customer behavior, top-selling products, and revenue by category.
